
# Letrify Frontend

Frontend da aplicação **Letrify** construído com Next.js (App Router), TypeScript e CSS (ou Tailwind).

> 🚀 Projeto criado com `create-next-app` em `letrify-front`.

## ✅ Início rápido

1. Instale dependências:

```bash
npm install
# ou
npm install
```

2. Execute em modo desenvolvimento:

```bash
npm run dev
# ou
npm dev
```

3. Abra no navegador:

[http://localhost:3000](http://localhost:3000)

## 📁 Estrutura principal

- app — rotas e páginas do App Router.
- `app/page.tsx` — página inicial.
- layout.tsx — layout global.
- globals.css — estilos globais.
- public — assets públicos.
- `next.config.mjs` — configurações do Next.js.
- tsconfig.json — configuração TypeScript.

## 🧭 Principais scripts

```bash
npm run dev       # server de dev
npm run build     # build de produção
npm run start     # rodar build
npm run lint      # ESLint
npm run typecheck # checagem TypeScript
```

## 📦 Deploy

1. Commit + push no repositório.
2. Conecte no Vercel.
3. Configure variáveis (`NEXT_PUBLIC_*`).
4. Deploy automático a cada push.
